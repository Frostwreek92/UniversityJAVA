Introducción a los Arreglos en Java
- Los arreglos (arrays) en Java son estructuras de datos que permiten almacenar múltiples valores del mismo tipo en una sola variable. Esta lección cubrirá la sintaxis básica para declarar, inicializar y manipular arreglos en Java, además de presentar algunos ejemplos y aplicaciones del mundo real.

1. Declaración e Inicialización de Arreglos
 1. Declaración de un arreglo:
  int[] numeros;
 2. Inicialización de un arreglo:
  numeros = new int[5]; // Un arreglo de 5 elementos
 3. Declaración e inicialización en una sola línea:
  int[] numeros = new int[5];
 4. Inicialización con valores específicos:
  int[] numeros = {1, 2, 3, 4, 5};

- Ejemplo Básico:
 public class ArreglosBasicos {
  public static void main(String[] args) {
   // Declaración y inicialización de un arreglo de enteros
   int[] numeros = {10, 20, 30, 40, 50};

   // Acceso a los elementos del arreglo
   System.out.println("El primer elemento es: " + numeros[0]); // Imprime 10
   System.out.println("El segundo elemento es: " + numeros[1]); // Imprime 20

   // Modificación de un elemento del arreglo
   numeros[2] = 100;
   System.out.println("El tercer elemento modificado es: " + numeros[2]); //Imprime 100
  }
 }

2. Recorrer Arreglos
- Para recorrer un arreglo, podemos utilizar un bucle for:
 1. Bucle for tradicional:
  for (int i = 0; i < numeros.length; i++) {
   System.out.println(numeros[i]);
  }
 2. Bucle for-each:
  for (int numero : numeros) {
   System.out.println(numero);
  }
- Ejemplo de Recorrido
 public class RecorrerArreglos {
  public static void main(String[] args) {
   // Declaración e inicialización de un arreglo de cadenas
   String[] nombres = {"Ana", "Juan", "Pedro", "María"};

   // Uso de bucle for tradicional
   System.out.println("Usando bucle for tradicional:");
   for (int i = 0; i < nombres.length; i++) {
    System.out.println(nombres[i]);
   }

   // Uso de bucle for-each
   System.out.println("\nUsando bucle for-each:");
   for (String nombre : nombres) {
    System.out.println(nombre);
   }
  }
 }

3. Aplicaciones del Mundo Real
- Ejemplo 1: Cálculo del Promedio de Calificaciones
 public class PromedioCalificaciones {
  public static void main(String[] args) {
   double[] calificaciones = {85.5, 90.3, 78.8, 92.0, 88.5};
   double suma = 0;

   for (double calificacion : calificaciones) {
    suma += calificacion;
   }

   double promedio = suma / calificaciones.length;
   System.out.println("El promedio de calificaciones es: " + promedio);
  }
 }
- Ejemplo 2: Encontrar el Valor Máximo en un Arreglo
 public class ValorMaximo {
  public static void main(String[] args) {
   int[] numeros = {34, 67, 23, 89, 12, 45};
   int maximo = numeros[0];
   for (int numero : numeros) {
    if (numero > maximo) {
     maximo = numero;
    }
   }

   System.out.println("El valor máximo en el arreglo es: " + maximo);
  }
 }
